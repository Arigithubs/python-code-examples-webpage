Python Code Examples

Example 1: Hello, World!

print("Hello, World!")

Example 2: Basic Arithmetic

x = 10
y = 5
print(x + y)  # Output: 15
print(x - y)  # Output: 5
print(x * y)  # Output: 50
print(x / y)  # Output: 2.0

Example 3: Working with Strings

name = "Bard"
greeting = "Hello, " + name + "!"
print(greeting)  # Output: Hello, Bard!

capitalized_name = name.capitalize()
print(capitalized_name)  # Output: Bard

Example 4: Conditional Statements

if x > y:
    print("x is greater than y")
else:
    print("x is less than or equal to y")

Example 5: Loops

for i in range(5):
    print(i)  # Output: 0, 1, 2, 3, 4
